This folder contains 2 files:
1) index.js
2) package


-The index.js contains the main code and logic.
-The package file contains dependencies and current information about the libraries and tools.

-We need to create a ".env" file and copy our PAT token in the .env file. Create a Variable name PAT and copy the token.
--The .env file contains the Personal Access Token(PAT). This token can be generated from your Azure DevOps Account. The token needs to be generated again when it is expired.
